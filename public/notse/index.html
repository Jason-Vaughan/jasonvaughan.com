Live teleprompter system, built for broadcast.

Notse is a real-time teleprompter for on-set production workflows. Cross-platform desktop app with live WebSocket sync between operator and talent — no cloud round-trip, no lag.

What it does

Real-time sync

WebSocket-based scroll, font, and content sync between operator and talent screen — sub-frame precision.

Cross-platform

Native Electron desktop app for macOS and Windows. No browser, no extensions, no cloud account.

Production-ready

Designed for the on-set workflow: hotkey-driven, mirror display, color presets, multi-script queue.

Closed source

Available under commercial license only. Source code is not distributed.
